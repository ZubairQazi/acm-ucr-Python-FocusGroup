# acm-ucr-Python-FocusGroup
Includes resources and coding challenges for the Python focus group by ACM at UCR. Feel free to clone and practice!

### JupyterNotebooks
Contains the Jupyter Notebook files for each topic covered in the focus group meetings. Each topic is separated by folders, and the solutions to the challenges are in different notebooks under the same directory.

### PythonChallenges
Contains the questions and solutions to the Python challenges from the focus group meetings. Each python file contains the challenge and its corresponding solution.
